# WP Releases

Scrape WordPress releases using Node.js

### What?

✔ Scrape releases from WP Release history
✔ Save data as JSON ``data.json``
✔ Download images with the Jazz Profiles using Google Image Search. (Only Public Domain images)

### Use

Run scraper

	node scraper.js

Miniy JPG images
	
	gulp
